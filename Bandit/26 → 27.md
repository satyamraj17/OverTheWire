# Level Goal

Good job getting a shell! Now hurry and grab the password for bandit27!

# Solution

We will need the shell we got when we escalated from level 25 to level 26. After that:
```
bandit26@bandit:~$ ls -l
total 20
-rwsr-x--- 1 bandit27 bandit26 14888 Apr  3 15:17 bandit27-do
-rw-r----- 1 bandit26 bandit26   258 Apr  3 15:17 text.txt

bandit26@bandit:~$ file bandit27-do 
bandit27-do: setuid ELF 32-bit LSB executable, Intel 80386, version 1 (SYSV), dynamically linked, interpreter /lib/ld-linux.so.2, BuildID[sha1]=fa7acfeea5c211d58e4f06864c5336d120073d41, for GNU/Linux 3.2.0, not stripped
```
SUID bit set for bandit27-do binary.
```
bandit26@bandit:~$ ./bandit27-do 
Run a command as another user.
  Example: ./bandit27-do id
bandit26@bandit:~$ ./bandit27-do whoami
bandit27
```
Run a command as any other user. Printing the content of the /etc/bandit_pass/bandit27
```
bandit26@bandit:~$ ./bandit27-do cat /etc/bandit_pass/bandit27
upsNCc7vzaRDx6oZC6GiR6ERwe1MowGB
```
