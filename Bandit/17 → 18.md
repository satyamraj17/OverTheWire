# Level Goal

There are 2 files in the homedirectory: passwords.old and passwords.new. The password for the next level is in passwords.new and is the only line that has been changed between passwords.old and passwords.new

NOTE: if you have solved this level and see ‘Byebye!’ when trying to log into bandit18, this is related to the next level, bandit19

# Solution
There is only one line difference between the two files: the password for level 18. The files are long, and a manual check will take ages. But Linux has our back. We can use the `diff` command, which compares files line by line and returns the differences.
```
bandit17@bandit:~$ diff passwords.old passwords.new
42c42
< 390zFj2NETFVZkqYw8UEFdN6h40oGVtT
---
> x2gLTTjFwMOhQ8oWNbMN362QKxfRqGlO
```
The bottom one is the password.
We won't be able to log in. We will see the message `Byebye !`. Onto the next challenge.
